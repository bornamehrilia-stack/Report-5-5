# Report-5-5



## ۱. مقدمه  
در این پروژه، یک **buzzer** به آردوینو متصل شده است و با استفاده از کد، یک ملودی ساده (مثل صدای تماس تلفن) پخش می‌شود. .

## ۲. اهداف پروژه  
- آشنایی با ساختار نت‌های موسیقی و فرکانس آن‌ها  
- یادگیری نحوه‌ی تولید صدا با `tone()`  
- استفاده از آرایه‌ها برای ذخیره نت‌ها و مدت زمان آن‌ها  
- پخش یک ملودی ساده  
- آشنایی با مفهوم “مدت زمان نت” و “فاصله بین نت‌ها”

## ۳. قطعات مورد استفاده  
- برد آردوینو Uno  
- یک buzzer (سیم‌پیچ صوتی)  
- برد برد (Breadboard)  
- سیم‌های جامپر (jumper wires)



## ۴. نحوه‌ی ساخت مدار  
- **buzzer**:  
  - یک سر → **پین دیجیتال ۹** آردوینو  
  - سر دیگر → **GND**

## ۵. کد برنامه :

```cpp
#include "pitches.h"  // اضافه کردن کتابخانه نت‌های موسیقی

// نت‌های ملودی "موبایل"
int mobile_notes[] = {
  NOTE_C4, NOTE_DS4, NOTE_G4, NOTE_C4, NOTE_DS4, NOTE_G4, NOTE_F4, NOTE_DS4, NOTE_F4, NOTE_DS4
};

// مدت زمان هر نت (به واحد ۱/۸)
int mobile_durations[] = {8, 8, 4, 8, 8, 8, 16, 16, 2};

void setup() {
  mobile();  // پخش ملودی
}

void loop() {
  // بدون کد — فقط یکبار اجرا می‌شود
}

void mobile() {
  for (int thisNote = 0; thisNote < 10; thisNote++) {
    int mobile_duration = 1000 / mobile_durations[thisNote];  // محاسبه مدت زمان نت
    tone(9, mobile_notes[thisNote], mobile_duration);          // پخش نت
    int pauseBetweenNotes = mobile_duration * 1.43;           // محاسبه فاصله بین نت‌ها
    delay(pauseBetweenNotes);                                 // تأخیر بین نت‌ها
  }
}
```

### توضیح کد:  
- `#include "pitches.h"` — اضافه کردن کتابخانه‌ی نت‌های موسیقی.  
- `mobile_notes[]` — یک آرایه است که نت‌های ملودی را ذخیره می‌کند.  
- `mobile_durations[]` — یک آرایه است که مدت زمان هر نت را ذخیره می‌کند.  
- `tone(9, mobile_notes[thisNote], mobile_duration)` — پخش نت با فرکانس مشخص و مدت زمان مشخص.  
- `delay(pauseBetweenNotes)` — تأخیر بین نت‌ها برای ایجاد اثر موسیقی.  
- `setup()` — فقط یکبار اجرا می‌شود و ملودی را پخش می‌کند.  
- `loop()` — خالی است — چون ملودی فقط یکبار پخش می‌شود.

## ۶. نحوه‌ی کارکرد  
- پس از آپلود کد، buzzer شروع به پخش ملودی “موبایل” می‌کند.  
- ملودی شامل ۱۰ نت است که هر کدام با مدت زمان مشخص پخش می‌شوند.  
- بعد از پخش ملودی، buzzer خاموش می‌شود.

## ۷. نتایج  
buzzer به درستی ملودی “موبایل” را پخش می‌کند. این نشان‌دهنده‌ی عملکرد صحیح ارتباط آردوینو با buzzer است.
